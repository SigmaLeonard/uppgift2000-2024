Molntjänster har förändrat IT genom att erbjuda företag flexibla och skalbara lösningar som sänker kostnader. Amazon Web Services (AWS), lanserad 2006, var först med att erbjuda företag att betala bara för den kapacitet de använder. Google Cloud Platform (GCP) kom 2008 och fokuserar på datalagring samt verktyg för maskininlärning och AI. Microsoft Azure, lanserad 2010, integrerar Microsofts tjänster och stödjer hybridlösningar för en enkel övergång till molnet. Dessa plattformar driver utvecklingen av Software as a Service (SaaS), vilket ger företag kostnadseffektivitet, säkerhet och möjlighet att växa utan att oroa sig för IT-infrastrukturen.
