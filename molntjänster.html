Molntjänster är tjänster som gör att företag kan lagra och hantera data över internet istället för på egna servrar. Detta gör det enklare och billigare för företag att använda teknologi, eftersom de slipper investera i dyr hårdvara och infrastruktur.

Dessa plattformar har gjort det lättare för företag att använda teknologi utan att behöva bekymra sig om kostnaderna för att köpa och underhålla hårdvara. Det ger dem en möjlighet att fokusera på att växa och innovera istället för att bara hantera IT-infrastrukturen.

- Amazon Web Services (AWS): Lanserades 2006 även först med molntjänster. Erbjuder mängder olika lösningar för företag.

- Google Cloud Platform (GCP): Introducerad 2008, känd för datalagring och avancerad teknik inom maskininlärning och AI.

- Microsoft Azure: Lanserades 2010, integreras lätt med Microsoft-tjänster och stöder hybridlösningar som kombinerar egna servrar med molnet.
